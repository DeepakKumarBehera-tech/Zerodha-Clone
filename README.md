# Zerodha Clone

A full-stack stock trading platform inspired by Zerodha, built using the MERN stack.

## Features

- User-friendly trading dashboard
- Holdings and positions management
- Orders section
- Responsive UI
- REST API integration
- MongoDB database connectivity

## Tech Stack

### Frontend
- React.js
- Material UI
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
